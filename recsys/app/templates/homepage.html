How the Recommender Works

This recommender engine uses collaborative filtering to profile similar users into clusters. Users in a cluster receive the same recommendations. Recommendations are in the form of which content appears first on a horizontal scrolling pane. We use text analysis to determine which events/movies should appear first to the user.
